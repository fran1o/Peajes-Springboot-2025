# Sistema de Gestión de Peajes en Ruta



## Descripción

Sistema de gestión desarrollado con Java y Spring Boot con arquitectura backend y lógica de negocio centralizada para la gestión y monitoreo de un sistema de peajes en ruta.

Permite la administración de propietarios y la simulación de eventos operativos con actualización de información en tiempo real mediante patrones de diseño.



## Funcionalidades

- Autenticación con roles (Administrador / Propietario)

- Gestión de propietarios del sistema

- Simulación de eventos de tránsito

- Asignación de bonificaciones

- Cambio de estado de propietarios

- Visualización de información en tiempo real

- Actualización automática de datos mediante Observer



## Tecnologías

- Java

- Spring Boot

- Maven

- Programación orientada a objetos

- Patrones de diseño (MVC, Facade, Observer, State)

- Arquitectura en capas



## Arquitectura

- Controllers (manejo de flujo de la aplicación)

- Services (lógica de negocio)

- Facade (punto central del sistema)

- Model (entidades del dominio)

- Observer (actualización en tiempo real)

- State (manejo de estados dinámicos)
